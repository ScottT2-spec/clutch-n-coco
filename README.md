# Clutch 'N' Coco

**Unbox Your Next Obsession**

Static landing page for Clutch 'N' Coco — an affordable fashion & accessories boutique serving UENR campus, Ghana.

## Stack

- **HTML5** — semantic markup, ARIA roles
- **CSS3** — custom properties, Grid/Flexbox layout, clamp-based responsive type
- **Vanilla JS** — IntersectionObserver scroll-reveal, mobile nav, no dependencies

## Fonts

- [Great Vibes](https://fonts.google.com/specimen/Great+Vibes) — brand script
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) — headings
- [Inter](https://fonts.google.com/specimen/Inter) — body text

## Local development

Open `index.html` in a browser, or serve with any static file server:

```bash
python3 -m http.server 8080
```

Hosted on [GitHub Pages](https://scottt2-spec.github.io/clutch-n-coco/).
